المميزات الأساسية

بساطة التكوين

عناصر HTML اعتيادية تتحول لسحر بمجرد استدعاء المكتبة.

تأثيرات مجنونة 3D

Cube, Flip, Scale-Rotate, Fade, Slide... كلها جاهزة للاستخدام.

دعم كامل

عجلة الفأرة، أسهم الكيبورد، والسحب باللمس على الجوال.

خفيف وسريع

مكتوب بـ Vanilla JS النقي بدون أي مكاتب خارجية (مثل jQuery).

الخطوة الأولى: الاستدعاء

قم بتضمين ملفات المكتبة في صفحتك، واحرص على وضع التنسيقات في قسم `head` والجافاسكريبت قبل نهاية `body`.

<!-- في الـ Head -->
<link rel="stylesheet" href="scrol.css">

<!-- قبل نهاية الـ Body -->
<script src="scrol.js"></script>

الخطوة الثانية: هيكل الصفحة

قم بإنشاء وعاء رئيسي وبداخله مجموعة من الأقسام `section`.

<div id="main-container">
    <section> المشهد الأول </section>
    <section> المشهد الثاني </section>
    <section> المشهد الثالث </section>
</div>

الخطوة الثالثة: التفعيل

استدعِ المكتبة وحدد الخيارات كما يناسبك:

<script>
    document.addEventListener('DOMContentLoaded', () => {
        const myScrol = new Scrol({
            container: '#main-container', // الوعاء الرئيسي
            selector: 'section', // وسم الأقسام
            effect: 'slide', // التأثير المفضل لك
            duration: 1000 // المدة بالميلي ثانية
        });
    });
</script>
