## نظرة عامة
**Scrol.js** هي مكتبة جافاسكريبت نقية (Vanilla JS) تهدف إلى تحويل صفحات الويب العادية إلى عروض تقديمية وتجارب بصرية مذهلة. بدون الحاجة لأي مكتبات خارجية، تمنحك المكتبة قدرة على التنقل بين أقسام موقعك باستخدام تأثيرات 3D احترافية وبأداء عالي.

<br />

## المميزات الأساسية
- **بساطة التكوين**: تحويل عناصر HTML الاعتيادية إلى تأثيرات مذهلة بمجرد استدعاء المكتبة.
- **تأثيرات 3D متقدمة**: مجموعة واسعة من التأثيرات الجاهزة مثل المكعب، الانقلاب، الدوران المدمج مع التصغير، والمزيد.
- **دعم شامل للأجهزة**: تحكم كامل عبر عجلة الفأرة، أسهم لوحة المفاتيح، والسحب باللمس للأجهزة المحمولة.
- **أداء فائق**: مبنية بالكامل باستخدام JavaScript النقي، خفيفة الوزن وسريعة الاستجابة.

<br />

## التثبيت والاستدعاء

الخطوة الأولى تتمثل في تضمين ملفات المكتبة داخل مشروعك. يجب وضع ملف التنسيقات `scrol.css` في وسم `<head>`، وملف الجافاسكريبت `scrol.js` قبل إغلاق وسم `<body>`.

```html
<!-- في قسم Head -->
<link rel="stylesheet" href="scrol.css">

<!-- قبل نهاية قسم Body -->
<script src="scrol.js"></script>
```

## كيفية الاستخدام

### 1. هيكل الصفحة HTML
قم ببناء هيكل الصفحة بوضع وعاء رئيسي يحتوي على مجموعة من الأقسام التي ستمثل مشاهد التمرير.

```html
<div id="main-container">
    <section>المشهد الأول</section>
    <section>المشهد الثاني</section>
    <section>المشهد الثالث</section>
</div>
```

### 2. التهيئة
بعد استدعاء الملفات وبناء الهيكل، قم بتهيئة المكتبة واختيار التأثير والمدة الزمنية التي تناسبك.

```javascript
document.addEventListener('DOMContentLoaded', () => {
    const myScrol = new Scrol({
        container: '#main-container', // معرف الوعاء الرئيسي
        selector: 'section',          // وسم أو صنف الأقسام الداخلية
        effect: 'slide',              // نوع التأثير المختار
        duration: 1000                // مدة التأثير بالميلي ثانية
    });
});
```

## التأثيرات المدعومة

توفر المكتبة 10 تأثيرات بصرية متنوعة يمكنك التبديل بينها عبر خيار `effect` عند التهيئة:

| اسم التأثير | الوصف البصري |
| :--- | :--- |
| `slide` | انزلاق عمودي كلاسيكي سلس |
| `fade` | تلاشي وظهور ناعم |
| `zoom` | تكبير المشهد وتصغيره |
| `cube` | انتقال على شكل دوران مكعب 3D |
| `flip` | انقلاب كامل للمشهد كبطاقة 3D |
| `scale-rotate` | دوران مدمج مع تصغير الحجم |
| `bounce` | وثب مرن ديناميكي |
| `swing` | تأرجح ثلاثي الأبعاد |
| `spin-out` | دوران لولبي حلزوني للخارج |
| `door` | فتح الأبواب للمشهد التالي |
